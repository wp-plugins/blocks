== Description ==

Simple and flexible content management block.


= Key Features =

* Easy to use
* reusable content blocks with [shortcodes]
* paste the generated [shortcodes] to anywhere you want the block content added.

= Please note*** =

* Support for categories - coming soon
* Support for tags - coming soon

= Requirements =

1. Self hosted WordPress.org installation (4.0 or higher)

== Installation ==

1. Upload the entire 'contact-form-7-mail-chimp-extension' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Blocks' menu item in the admin manu.
